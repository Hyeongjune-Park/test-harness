# Test Harness Playground

Small toy project for testing global planner and reviewer agents.

## Purpose

This project is intentionally simple and slightly messy.
It exists to test how planning and review agents analyze:
- feature requests
- refactoring ideas
- validation needs
- scope control

## Current state

- Basic item list rendering exists
- Filtering/search does not exist
- Validation is weak
- Some values are hardcoded
- Utility functions are inconsistent

## Possible future tasks

- Add simple search
- Add input validation
- Refactor duplicate logic
- Move hardcoded values into config
- Improve rendering structure
