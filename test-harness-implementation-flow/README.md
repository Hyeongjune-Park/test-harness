# Task API Mini Project

A minimal Express + TypeScript project for testing harness workflows.

## Current Features

- Create a task
- List tasks

## Existing Routes

- `POST /tasks`
- `GET /tasks`

## Data Shape

A task currently contains:
- `id`
- `title`
- `completed`

## Notes

- There is currently no endpoint for marking a task as complete.
- Validation is intentionally simple.
- Storage is in-memory only.
